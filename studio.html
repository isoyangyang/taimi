How to publish it (about 1 minute)

After you download the file, put it on your website like this:

  1. Go to your project on github.com and sign in.
  2. Open the content folder.
  3. Click Add file → Upload files (top right).
  4. Drag the file you just downloaded into the box. Because it has the same name, it replaces the old one.
  5. Scroll down and click the green Commit changes button.

Your site updates by itself in a minute or two. Refresh the app to see the new content. The file you download already contains everything that's there now plus what you added, so nothing gets lost.
